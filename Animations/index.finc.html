Scroll Animations

A variety of animations triggered on scroll (powered by WOW/animate.css)

The following information is furnished for historical purposes only and may be removed. New sites should use ScrollEffects instead of WOW/Animate.css as the latter imposes a large file size burden onto every site that uses it.

Existing sites based on the Phylactery framework should ensure that Animate.css is not present if it is not in use and consider migrating away from it as soon as possible.

These animations are useful for calls to action. The primary class used is .wow to trigger the scroll animation.
