Single-Column Affix

        
This is an affix column setup with only one column. You need to manually specify a height on the relative wrapper element (usually, AffixColumn-root) and then apply the attribute data-affixcolumn-options="noheightbearing" in order to disable the automatic height-bearing detection. This effect can be useful for sections which are recieving their own scroll effects.
